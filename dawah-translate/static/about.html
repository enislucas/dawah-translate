About Dawah-Translate V3

A self-hosted pipeline that translates Islamic YouTube content into Romanian, producing ready-to-upload videos with burned-in subtitles.

The Problem

There is a wealth of Islamic educational content on YouTube in Arabic and English, but very little is accessible to Romanian-speaking Muslims. Manually translating and subtitling videos is extremely time-consuming. Existing tools like YouTube's auto-translate produce poor results for Islamic terminology -- translating "Salah" and "Du'a" to the same Romanian word ("rugaciune"), losing critical theological distinctions.

Problem

Arabic Islamic terms get mistranslated. "Salah" (ritual prayer) and "du'a" (supplication) both become "rugaciune" in Romanian, erasing the distinction.

Solution

A glossary system that transliterates terms (Salah, du'a) and adds explanations only on first occurrence: "Salah (rugaciunea rituala)".

Problem

AI translations produce "walls of text" -- 15-word subtitles that stay on screen for 1.5 seconds. Impossible to read.

Solution

Reading speed validation (max 21 characters/second), automatic resegmentation, and a review UI that highlights too-fast segments.

Problem

Romanian diacritics (ș, ț) break in many subtitle renderers. AI models often output the wrong Unicode variant (cedilla instead of comma-below).

Solution

Automatic normalization of all ş→ș and ţ→ț throughout the pipeline, plus Noto Sans font for correct glyph rendering.

How It Works

The V3 pipeline has 7 stages, with a human-in-the-loop config step and a 4-stage translation engine:

1. Download

Paste a YouTube URL. The system downloads the video (max 1080p) and extracts audio optimized for speech recognition (16kHz mono WAV).

2. Transcribe

Audio is transcribed using Whisper (faster-whisper, CPU). Supports Arabic, English, Turkish, French, and auto-detection. Produces timestamped SRT subtitles.

3. Configure

Pipeline pauses. A config screen shows video info, cost estimate per model (Sonnet/Opus with EUR ranges), and subtitle mode selector (black bar default vs overlay).

4. Translate (4-stage pipeline)

Stage 1 — Analysis: Full-document summary, glossary term detection, transcription error flagging, Romanian title generation.
Stage 2 — Raw Translation: Sliding-window translation (50 segments, 15 overlap) with short accuracy-focused prompt.
Stage 3 — Quality Review: Polish pass with a separate short prompt for naturalness and diacritics.
Stage 4 — Validation: Python-based checks (reading speed, diacritics, glossary compliance) — no LLM cost.

5. Review

Full review UI: edit translations inline, edit timestamps with keyboard shortcuts ([/]), per-segment feedback (good/bad), re-translate individual segments, editable Romanian title. Transcription errors highlighted in red.

6. Burn

FFmpeg burns subtitles into the video. Black bar mode adds a clean 160px strip below the video so Romanian text never overlaps existing English/Arabic subtitles. Noto Sans Bold with outline.

7. Download

Download the final MP4, named with the Romanian title for easy identification.
