About Dawah-Translate

A self-hosted pipeline that translates Islamic YouTube content into Romanian, producing ready-to-upload videos with burned-in subtitles.

The Problem

There is a wealth of Islamic educational content on YouTube in Arabic and English, but very little is accessible to Romanian-speaking Muslims. Manually translating and subtitling videos is extremely time-consuming. Existing tools like YouTube's auto-translate produce poor results for Islamic terminology -- translating "Salah" and "Du'a" to the same Romanian word ("rugaciune"), losing critical theological distinctions.

Problem

Arabic Islamic terms get mistranslated. "Salah" (ritual prayer) and "du'a" (supplication) both become "rugaciune" in Romanian, erasing the distinction.

Solution

A glossary system that transliterates terms (Salah, du'a) and adds explanations only on first occurrence: "Salah (rugaciunea rituala)".

Problem

AI translations produce "walls of text" -- 15-word subtitles that stay on screen for 1.5 seconds. Impossible to read.

Solution

Reading speed validation (max 21 characters/second), automatic resegmentation, and a review UI that highlights too-fast segments.

Problem

Romanian diacritics (ș, ț) break in many subtitle renderers. AI models often output the wrong Unicode variant (cedilla instead of comma-below).

Solution

Automatic normalization of all ş→ș and ţ→ț throughout the pipeline, plus Noto Sans font for correct glyph rendering.

How It Works

The pipeline has 5 main stages:

1. Download

Paste a YouTube URL. The system downloads the video (max 1080p) and extracts audio optimized for speech recognition (16kHz mono WAV).

2. Transcribe

Audio is transcribed using OpenAI's Whisper model (running locally via faster-whisper). Supports Arabic, English, Turkish, French, and auto-detection. Produces timestamped SRT subtitles.

3. Translate

Subtitles are translated to Romanian. Choose between free local translation (NLLB-200, Meta's multilingual model) or Claude API (higher quality, understands Islamic context). Both apply the glossary.

4. Review

A web UI shows the video alongside the subtitle table. Edit translations inline, check reading speed, re-translate individual segments. The human is in the loop before burning.

5. Burn & Download

Once approved, FFmpeg burns the subtitles directly into the video as styled text (Noto Sans Bold, white with black outline). Download the final MP4.

Project Roadmap

COMPLETE

Phase 1: Core Pipeline

Download, transcribe, translate, review, and burn subtitles. Full web UI with job management.

NEXT

Phase 2: Quality & Polish

Speaker diarization, translation memory, Telegram notifications, batch processing, Google Drive upload.

FUTURE

Docker deployment, multi-user auth, YouTube direct upload, cloud hosting options.

Expansion Ideas

  • Quran verse detection -- Detect Quranic recitation, auto-insert standard Romanian translation
  • Islamic lecture database -- Index translated lectures by topic, speaker, difficulty
  • Community review -- Multiple reviewers collaborating on translations
  • A/B translation testing -- Compare two translations, pick the better one
  • Audio dubbing -- Generate Romanian audio track via TTS (experimental)
  • Subtitle style presets -- Different styles for lectures, debates, nasheed

Who Made This

This project was built for the Romanian Muslim community to make Islamic educational content more accessible. It's open source and welcomes contributions.

Built with the help of Claude (Anthropic) for both the translation engine and the codebase itself.
